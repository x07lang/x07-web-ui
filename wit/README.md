# WIT packages

This repo is the canonical home for the `x07:web-ui` WIT packages.

- Legacy contract line: `x07:web-ui@0.1.0` (legacy file: `wit/x07-web-ui.wit`)
- Current contract line: `x07:web-ui@0.2.0` (`wit/x07/web_ui/0.2.0/web-ui-app.wit`)

The current device-helper expansion in `std-web-ui@0.2.4` ships through the JSON effect/dispatch schemas, the stdlib package modules, and the host bridge assets. This release line does not introduce an `x07:web-ui@0.3.0` WIT package.
